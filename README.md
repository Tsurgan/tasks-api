API для управления задачами на Laravel

Создание задачи: POST /tasks (поля: title, description, status).
Просмотр списка задач: GET /tasks (возвращает все задачи).
Просмотр одной задачи: GET /tasks/{id}.
Обновление задачи: PUT /tasks/{id}.
Удаление задачи: DELETE /tasks/{id}.
База данных SQLite находится в папке database.

Также функционал повторяется на Filament, его можно увидеть перейдя по ссылке "Перейти в панель Filament". Email и пароль тестового пользователя можно найти в базе данных.
